# Удивительный UX

Коллекция ссылок, материлов, видео по разнообразным аспектам UX. Под UX тут подразумевается не просто взаимодействие внутри приложения, а весь тот опыт, эмоции и впечатления, которые может принести приложение.

Рассматривается UX с точки зрения дизайна продукта, от вовлечения пользователя, через процесс использования, до получения результата, эмоций и опыта. То есть тут могут быть материалы по психологии, маркетингу и дизайну - все то, что и формирует UX пользователя.
